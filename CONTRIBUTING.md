# Beitrag zu n8n-business-automation

Vielen Dank für dein Interesse! Hier ein paar kurze Richtlinien.

## Workflow einreichen

1. Fork dieses Repo
2. Erstelle einen neuen Workflow in `workflows/`
3. Der Workflow muss als gültige n8n-JSON-Datei vorliegen
4. Erstelle einen Pull Request

## Workflow-Richtlinien

- **Keine echten API-Keys** im Workflow — nur Platzhalter
- **Dokumentation:** Füge eine kurze README-Sektion hinzu, die erklärt:
  - Was der Workflow macht
  - Welche Services/Credentials benötigt werden
  - Wie er konfiguriert wird
- **Sprache:** Deutsch (DACH-Zielgruppe)
- **Lizenz:** MIT

## Naming-Konvention

Dateien: `kebap-case.json`, z.B. `kundendatenbank-sync.json`
